Website

Durante as aulas de Fundamentos de Desenvolvimento de Software no curso de Análise e Desenvolvimento de Sistemas, desenvolvi um projeto prático voltado à criação de um website básico. O objetivo da atividade foi aplicar os conceitos iniciais de desenvolvimento web, compreendendo a estruturação de uma página e sua publicação na internet.

O projeto consistiu na construção de uma página única (single page), contendo informações organizadas de forma clara e objetiva. Para isso, utilizei HTML para a estruturação do conteúdo e CSS para a estilização, buscando criar um layout simples, porém funcional e visualmente agradável.

Durante o desenvolvimento, foram trabalhados conceitos fundamentais como organização de elementos na página, uso de títulos, parágrafos, links e aplicação de estilos básicos. Além disso, foi possível compreender a importância da padronização do código e da separação entre estrutura e apresentação.

Após a finalização, o site foi publicado utilizando a plataforma GitHub, permitindo que o projeto ficasse disponível online. Esse processo contribuiu para o entendimento inicial sobre versionamento de código e hospedagem de páginas web.

A atividade foi essencial para consolidar os primeiros conhecimentos em desenvolvimento web, proporcionando uma visão prática de como criar, organizar e disponibilizar um site na internet.
