Banco de Dados

Durante as aulas de Banco de Dados Relacional no curso de Análise e Desenvolvimento de Sistemas, desenvolvi um projeto voltado à criação e implementação de um banco de dados utilizando MySQL. O objetivo da atividade foi aplicar conceitos fundamentais relacionados à modelagem, estruturação e manipulação de dados.

O projeto envolveu a criação de tabelas, definição de atributos, estabelecimento de chaves primárias e estrangeiras, além da organização dos relacionamentos entre as entidades. Esses elementos foram essenciais para garantir a integridade e a consistência das informações armazenadas no banco de dados.

Após a implementação da estrutura, foram realizadas diversas consultas utilizando a linguagem SQL, com o intuito de testar o funcionamento do banco e validar os dados inseridos. As pesquisas incluíram comandos de seleção, filtragem e organização das informações, permitindo verificar na prática a eficiência do modelo desenvolvido.

Durante o desenvolvimento, foram aplicados conceitos importantes como normalização de dados, integridade referencial e boas práticas na construção de consultas. A atividade também contribuiu para o aprimoramento do raciocínio lógico e da capacidade de resolver problemas relacionados ao armazenamento e recuperação de dados.

Esse projeto foi fundamental para consolidar os conhecimentos em bancos de dados, proporcionando uma visão prática sobre como estruturar e manipular informações de forma eficiente em sistemas reais.
